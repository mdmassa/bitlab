# BitLab 🐧

Opa, beleza? Este repositório é um acompanhamento das minhas atividades, aprendizados e projetos em linguagem C. Estou buscando compreender melhor a parte avançada da linguagem e conceitos mais complexos, aprofundando o que foi aprendido na graduação em engenharia da computação.

Voltar a linguagem C é um jeito de manter a lógica afiada e entender entrissicamente o que acontece por trás de tudo (seja em sistemas operacionais que utilizo ou em firmwares de projetos de automação que faço), e aqui será meu diário de bordo.

Estou organizando os códigos por tópicos e sempre manterei atualizado com revisões e desafios que eu resolver em cada aprendizado.

##### Como rodar o código

```
gcc arquivo.c -o executavel     // compilar

./executavel          // executar
```
